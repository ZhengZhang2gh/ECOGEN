Geometrical constructor from a XML format reading.

Reading data from XML file under the following format: ex : <dataRectangle axis1 = "x" axis2 = "y" lAxis1 = "0.3" lAxis2 = "0.2"> <posInferiorVertex x = "0.4" y = "0.5" z = "0."> </dataRectangle>
